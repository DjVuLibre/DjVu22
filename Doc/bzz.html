bzz

General purpose compression/decompression program
Compression:
bzz -e[<blocksize>] <infile> <outfile>
Decompression:
bzz -d <infile> <outfile>

Program bzz is a simple front-end for the Burrows Wheeler encoder implemented in BSByteStream.h. Although this compression model is not currently used in DjVu files, it may be used in the future for encoding textual data chunks. Argument blocksize is expressed in kilobytes and must be in range 200 to 4096. The default value is 2048. Arguments infile and outfile are the input and output filenames. A single dash ("-") can be used to represent the standard input or output.
