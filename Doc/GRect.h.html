GRect.h

Rectangle manipulation class.
GRect
Rectangle class.
GRectMapper
Maps points from one rectangle to another rectangle.
Files "GRect.h" and "GRect.cpp" implement basic operations on rectangles. Class GRect is used to represent rectangles. Class GRectMapper represent the correspondence between points relative to given rectangles. Class GRatio is used to represent scaling factors as rational numbers.
