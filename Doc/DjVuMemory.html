DjVu Memory

This section is enabled when compilation symbol NEED_DJVU_MEMORY is defined. Function _djvu_memory_callback can be used to redefine the C++ memory allocation operators. Some operating systems (e.g. Macintoshes) require very peculiar memory allocation in shared objects. We redefine the operators new and delete as want to export these redefined versions to other libraries.
