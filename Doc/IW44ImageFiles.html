IW44 Image Files

The IW44 Image file format is the native format for the IW44 wavelet representation. The DjVu Reference Library comes with two small programs, c44 and d44, for encoding and decoding IW44 Files.

Gray Level IW44 Images --- Gray Level IW44 Image files are composed of a single "FORM:BM44" composite chunk. This composite chunk itself contains one or more "BM44" chunks containing successive refinements of the gray level image (see IWBitmap).

Color IW44 Images --- Color IW44 Image files are composed of a single "FORM:PM44" composite chunk. This composite chunk itself contains one or more "PM44" chunks containing successive refinements of the color image (see IWPixmap).
